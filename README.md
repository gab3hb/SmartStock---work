# SmartStock QR

Sistema de controle logístico inteligente para gerenciamento de estoque por localização física utilizando QR Codes.

## Objetivo

Automatizar o processo de contagem e controle de cadeiras armazenadas em diferentes posições físicas do centro de distribuição, eliminando conferências manuais e aumentando a rastreabilidade operacional.

## Funcionalidades

* Autenticação com controle de permissões
* Cadastro de locais de armazenagem
* Geração automática de QR Codes por localização
* Leitura de QR Code via celular
* Consulta instantânea de estoque por local
* Atualização de quantidades em tempo real
* Histórico completo de movimentações
* Dashboard gerencial de estoque

## Stack Tecnológica

* Frontend: React / Next.js
* Backend: Node.js / Express
* Banco de Dados: PostgreSQL
* ORM: Prisma
* Autenticação: JWT
* QR Code: html5-qrcode / qrcode

## Público-Alvo

Operações logísticas, centros de distribuição e empresas com necessidade de rastreamento rápido de estoque físico por localização.
